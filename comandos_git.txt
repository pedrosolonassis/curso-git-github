GIT STATUS: Saber como está o diretório (se tem save)
GIT ADD: Adicionar o que levar para o commit
GIT COMMIT -M “nome” : Nomear o commit e criar o save
- GIT LOG: Mostra o histórico de commit
- GIT INIT: Novo repositório (Pasta especial = Repositório de GIT)
- GIT DIFF: Mostra as novas alterações no arquivo
